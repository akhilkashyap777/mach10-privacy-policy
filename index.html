Privacy Policy for Mach-10

Last Updated: August 16, 2025

Effective Date: August 16, 2025

No Data Collection Statement

Mach-10 does NOT collect, store, or transmit any personal data or information from users.

How Mach-10 Works

Mach-10 is a local file sharing application that enables users to share files between devices on their local network. The application operates entirely within your local network environment and does not require an internet connection to function.

What Data We Do NOT Collect

  • Personal information (names, email addresses, phone numbers)
  • File contents or file names
  • Device information or identifiers
  • Usage statistics or analytics
  • IP addresses or network information
  • Location data
  • Any other personal or technical data

Local Network Operation

All file transfers performed by Mach-10 occur directly between your devices on your local network. No data is sent to external servers, cloud services, or third parties. The application does not connect to the internet for any data collection or transmission purposes.

No Third-Party Services

Mach-10 does not integrate with or use any third-party analytics, advertising, or data collection services. The application operates as a standalone tool without external dependencies for data processing.

Data Security

Since Mach-10 does not collect or store any data, there are no data security concerns related to personal information. All file transfers remain within your local network and under your direct control.

Children's Privacy

Mach-10 does not collect any information from users of any age, including children under 13. The application can be safely used by users of all ages without privacy concerns.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time to reflect changes in our application or legal requirements. Any changes will be posted at this location with an updated "Last Updated" date. Since the application does not collect data, changes will typically be minor clarifications or legal updates.

Microsoft Store Compliance

This privacy policy is provided to comply with Microsoft Store requirements. Mach-10's privacy-first design ensures user data protection by not collecting any data in the first place.
